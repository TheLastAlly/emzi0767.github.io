Emzi0767's Discord bots

Here you can find all of my currently maintained Discord bots. They fulfill various tasks. There are some administrative bots, some niche ones, and even selfbots.

To show information about a bot, go to its page. There you will find invite links, documentation, and source code.

ADA

ADA (Automated/Advanced Discord Administrator) is designed to extend and simplify a number of administrative tasks. It's written in C#, using Discord.NET library.

PAM

PAM (Python Administrative Module) is a Python clone of ADA, written using Discord.py library.

ASM

ASM (Assembler) is a bot that assembles assembly code to machine code, and outputs the resulting hex. Written using Discord.py library.

DEVI

DEVI (Dynamic EValuation Implement) is a selfbot, designed with quoting and C# evaluation in mind. Written using Discord.NET library.
