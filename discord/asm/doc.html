Assembling code

@Assembler assemble <architecture> <code block> Assembles code. Architecture can be one of:

  • x86: x86 i386 i486 i586 i686
  • x64: x64 x86_64 amd64
  • x86 (AT&T syntax): x86_att i386_att i486_att i586_att i686_att
  • x64 (AT&T syntax): x64_att x86_64_att amd64_att
  • ARMv6: armv6 armv6k
  • ARMv6 (big-endian): armv6eb armv6keb armv6_be armv6k_be
  • ARMv7: armv7 armv7a
  • ARMv7 (big-endian): armv7eb armv7aeb armv7_be armv7a_be
  • ARMv8: armv8 armv8a
  • ARMv8 (big-endian): armv8eb armv8aeb armv8_be armv8a_be
  • AArch64 (64-bit ARMv8): aarch64 arm64
  • AArch64 (64-bit ARMv8, big-endian): aarch64eb arm64eb aarch64_be arm64_be
  • MIPS: mips
  • MIPS (little-endian): mipsel mips_le
  • PowerPC: powerpc powerpc32 ppc ppc32
  • PowerPC64: powerpc64 ppc64
  • PowerPC64 (little-endian): powerpc64el ppc64el powerpc64_le ppc64_le
